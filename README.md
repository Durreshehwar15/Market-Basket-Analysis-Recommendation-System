# 🛒 Market Basket Analysis and Product Recommendation System

## Overview

This project applies Market Basket Analysis and Association Rule Mining techniques to identify purchasing patterns within retail transaction data. Using the Apriori Algorithm, the system discovers products frequently purchased together and generates association rules that can support recommendation systems, cross-selling strategies, and inventory planning.

In addition to traditional association rule mining, the project includes an interactive rule exploration interface that allows users to dynamically filter, search, and analyze product relationships based on support, confidence, and lift metrics.
The project demonstrates how data analytics can be used to transform raw transaction data into actionable business insights for retail decision-making.

## Objectives

- Discover frequently purchased product combinations.
- Analyze customer purchasing behavior.
- Generate product recommendations using association rules.
- Visualize relationships between products.
- Build an interactive analytics interface for rule exploration.
- Support retail decision-making through data-driven insights.
